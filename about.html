Project : Alpha Flight API

  The project Alpha Flight API is a machine learning application interface being developed by a group of students studying Computer Engineering at the Engineering Faculty of Istanbul Kultur University. The project aims to develop a machine learning program to predict airline delays and cancellations. It is a project assigned for CSE6064-Software Engineering Lecture. The group of 4 students developing this project made this website to update their instructor on the project on a weekly basis. Yet anyone is free to check out how the project is going and learn from what we experience completing this project.
