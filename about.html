Project : Alpha Flight API

    The project Alpha Flight API is a machine learning application interface being developed by a group of students studying Computer Engineering at the Engineering Faculty of Istanbul Kultur University. The project aims to develop a machine learning program to predict airline delays and cancellations. It is a project assigned for CSE6064-Software Engineering Lecture. The group of four students developing this project made this website to update their instructor on the project on a weekly basis. Yet anyone is free to check out how the project is going and learn from what we experience completing this project.

    TL;DR: You are free to witness the progress of these four souls on a path to become better engineers and get a good grade. Maybe you, the reader will learn from our experience to become a better engineer. So I hope that our experience may be beneficial to all young engineers out there trying to master the art of engineering.
